## 미들웨어(Middleware)
- WAS(Web Application Server)
- RPC(Remote Procedure Call)
- ORB(Object Request Broker)
- TP monitor: 트랜잭션이 올바르게 처리되고 있는지 데이터를 감시하고 제어


```bash
# 20년 3회 필기
5. 미들웨어 솔루션의 유형에 포함되지 않는 것은?
① WAS ② Web Server
③ RPC ④ ORB
```

```bash
# 20년 2회 필기
9. 트랜잭션이 올바르게 처리되고 있는지 데이터를 감시하고 제어하는 미들웨어는?
① RPC        ② ORB
③ TP monitor ④ HUB
```
