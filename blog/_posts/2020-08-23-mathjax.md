Jekyll에 수학식을 표기하려면 MathJax를 연동해야한다.

1. _layout/post.html에서 `<article>` 다음에 아래 코드 추가
    ```html
    <!-- _layouts/post.html -->
    <article>
    {% raw %}
    {% include mathjax_support.html %}
    {% endraw %}
    ```

2. 테스트
    ```
    {% raw %}
    $$\sum_{i=0}^n i^2 = \frac{(n^2+n)(2n+1)}{6}$$
    {% endraw %}
    ```
    $$\sum_{i=0}^n i^2 = \frac{(n^2+n)(2n+1)}{6}$$
